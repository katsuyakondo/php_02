# マッチングアプリ風

## DEMO

- デプロイしている場合は URL を記入（任意）

## 紹介と使い方

- 情報入力し登録するとデータベースに登録される。
- マッチング画面では現状では名前と画像が表示されるようにしたかった。

## 工夫した点

- プロフィール画像をアップロードして、そてをデータベースから呼び出すこと。

## 苦戦した点

- 画像の表示させるのができていない。
